

Real::Enocde - Perl-Win32 Perl interaction with Progressive Networks
    ReadEncoder(tm).

Version: 0.04Beta


DESCRIPTION
This module allows for interaction with the RealEncoder, and thus the manipulation
of RealMedia files, and encoding to RealMedia format.
